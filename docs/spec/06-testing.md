### 6.3 边界测试

| 测试场景 | 输入 | 预期结果 |
|----------|------|----------|
| 身高超限 | 50cm 或 300cm | 提示范围错误，要求重新输入 |
| 体重超限 | 10kg 或 500kg | 提示范围错误，要求重新输入 |
| 目标不合理 | 1周减10kg | 提示安全减重范围，建议调整 |
| 连续发送图片 | 5张图片 | 依次处理，汇总显示 |
| 特殊字符 | "今天吃了🍚🍚" | 正确处理表情符号 |

### 6.4 性能测试

**指标**:
| 操作 | 目标响应时间 |
|------|-------------|
| 图片识别 | < 10 秒 |
| 热量计算 | < 1 秒 |
| 进度查询 | < 2 秒 |
| 数据写入 | < 500ms |

### 6.5 错误恢复测试

| 场景 | 模拟方式 | 预期结果 |
|------|----------|----------|
| 数据库文件损坏 | 删除 foods_merged.json | 提示数据缺失，不影响其他功能 |
| 日志文件损坏 | 损坏 JSON 格式 | 创建新日志文件 |
| API 不可用 | 断开网络 | 提示服务暂时不可用 |
