# 🦞 Easy Slim - 微信减肥助手

> 拍照就能记录饮食，自动计算热量，帮你科学减肥！

---

## 🎯 这是什么？

Easy Slim 是一个**减肥助手**，你可以通过微信/QQ 和它聊天：

- 📸 **拍照记录** - 拍张食物照片，自动识别并计算热量
- ⏰ **定时提醒** - 每天提醒你吃饭、运动，不怕忘记
- 📊 **进度追踪** - 随时查看减肥进度，了解自己吃了多少
- 💬 **智能建议** - 根据你的情况给出饮食建议

## 🔧 准备工作

在安装之前，你需要准备这些东西：

### 必需的东西

| 序号 | 准备内容 | 说明 | 获取方式 |
|------|----------|------|----------|
| 1️⃣ | **QClaw 软件** | 腾讯出品的 AI 助手软件 | [点击下载](https://qclaw.qq.com/) |
| 2️⃣ | **微信账号** | 用于和 AI 聊天 | 你已有的微信 |
| 3️⃣ | **AI 模型账号** | 用于识别图片中的食物 | 见下方说明 |

### AI 模型账号说明

> **为什么需要 AI 模型账号？**
> Easy Slim 需要识别你发的食物图片，这个识别能力来自 AI 模型（就像手机的语音助手）。
> QClaw 本身不内置这个能力，需要你配置一个。

**推荐的 AI 模型（选一个就行）：**

| 模型名称 | 特点 | 价格 | 推荐指数 |
|----------|------|------|----------|
| Claude（克劳德） | 识别准确，支持中文 | 有免费额度 | ⭐⭐⭐⭐⭐ |
| GPT-4 Vision | 识别能力强 | 需付费订阅 | ⭐⭐⭐⭐⭐ |
| 通义千问 | 国产模型，免费 | 免费 | ⭐⭐⭐⭐ |

**如何获取 Claude 账号：**

1. 访问 https://console.anthropic.com/
2. 注册账号（需要国外手机号或邮箱）
3. 创建 API Key（一串像 `sk-ant-xxx` 的字符）
4. 复制保存这个 Key，后面要用

## 📥 安装步骤

### 第一步：安装 QClaw

```
1. 访问 https://qclaw.qq.com/
2. 点击「立即下载」
3. 下载后双击安装
4. 安装完成，打开 QClaw
```

### 第二步：绑定微信

```
1. 打开 QClaw
2. 点击右上角「设置」⚙️
3. 选择「微信绑定」
4. 用微信扫描屏幕上的二维码
5. 微信显示「绑定成功」即可
```

**绑定成功后，你的微信会多一个「QClaw 助手」好友**

### 第三步：配置 AI 模型

```
1. 在 QClaw 中，进入「设置」⚙️
2. 选择「模型配置」
3. 点击「添加模型」
4. 选择「Claude」或你选择的模型
5. 粘贴你的 API Key
6. 点击「保存」
```

### 第四步：安装 Easy Slim 技能

**方式一：下载安装包（推荐新手）**

```
1. 在本页面点击「下载 ZIP」
2. 解压下载的文件到任意文件夹（比如 D:\easy-slim-skill）
3. 记住这个文件夹路径
```

**方式二：Git 克隆（适合开发者）**

```bash
git clone https://github.com/your-username/easy-slim-skill.git
```

### 第五步：在 QClaw 中添加技能

```
1. 打开 QClaw，进入「设置」⚙️
2. 选择「技能管理」
3. 点击「添加技能路径」
4. 选择刚才解压的文件夹（比如 D:\easy-slim-skill）
5. 点击「确定」
6. 重启 QClaw
```

## ❓ 常见问题

### Q1: 发送图片后没反应？

**可能原因和解决方法：**

| 原因 | 解决方法 |
|------|----------|
| AI 模型没配置好 | 检查 QClaw 设置中的模型配置 |
| API Key 过期或无效 | 重新获取并配置 API Key |
| 网络连接问题 | 检查网络是否正常 |
| 图片太小或模糊 | 重新拍一张清晰的照片 |

**解决步骤：**
```
1. 打开 QClaw 设置 → 模型配置
2. 点击「测试连接」
3. 如果测试失败，重新配置 API Key
```

### Q2: 图片识别不准确？

**可能情况：**

| 情况 | 示例 | 解决方法 |
|------|------|----------|
| 份量估算偏差 | 识别100g，实际150g | 回复正确份量，如「其实是200g」 |
| 食物识别错误 | 识别成炒饭，实际是炒面 | 回复正确名称，如「是炒面」 |
| 漏识别食物 | 漏了桌角的饮料 | 补充说明，如「还有一杯可乐」 |

### Q3: 为什么有些食物查不到热量？

Easy Slim 的食物数据库包含 **1800+ 种食物**，主要是常见的中国菜和主食。

如果遇到查不到的食物：
```
你: [发送螺蛳粉照片]

QClaw 助手:
🤔 数据库暂无「螺蛳粉」的精确数据

相近替代：
• 米粉 - 350 kcal/份
• 河粉 - 380 kcal/份

你可以：
1. 选择替代食物 - 直接回复食物名
2. 手动输入热量 - 回复「螺蛳粉 400kcal」
3. 跳过此项 - 回复「跳过」
```

### Q4: 定时提醒没有收到？

**检查清单：**

```
□ QClaw 软件是否在运行？
□ 电脑是否处于休眠状态？
□ 微信是否正常绑定？
□ 提醒时间是否设置正确？

排查步骤：
1. 确认 QClaw 在任务栏有图标
2. 测试微信绑定：发送「你好」看是否回复
3. 查看提醒时间：发送「查看我的设置」
```

### Q5: 如何更换提醒语气？

有三种语气可选：

| 语气 | 示例 |
|------|------|
| **温柔** 💗 | "亲爱的，该运动啦～加油！" |
| **严厉** 💪 | "快点去运动！别偷懒！" |
| **幽默** 🦞 | "再不运动就要变成小龙虾了！" |

```
你: 更换提醒语气为幽默

QClaw 助手:
✅ 已更换为幽默风格！

试试发送「测试提醒」感受新语气～
```

## 🔒 数据安全

### 你的数据存在哪里？

```
所有数据都存在你的电脑本地！

📁 存储位置：
你的电脑 → D:\easy-slim-skill\data\
                        ├── profile.json  （你的个人信息）
                        └── logs\         （每日记录）
                            └── 2026-05-04.json
```

### 数据安全保证

| 安全项 | 说明 |
|--------|------|
| ✅ 本地存储 | 所有数据存在你的电脑，不上传到云端 |
| ✅ 隐私保护 | 身高、体重等敏感信息不会公开 |
| ✅ 开源透明 | 代码完全公开，可以自己检查 |
| ✅ 不收集数据 | 开发者无法看到你的任何数据 |

### 哪些数据不会公开？

本项目的 `.gitignore` 配置确保以下数据**不会上传到 GitHub**：

```
❌ 不会上传的数据：
   • 你的身高、体重、年龄
   • 你的饮食记录
   • 你的运动记录
   • 你的体重变化数据
   • 你的微信/QQ 账号信息

✅ 可以上传的数据：
   • 食物营养数据库（公开数据）
   • 提醒消息模板（不含个人信息）
```
